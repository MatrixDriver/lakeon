# DBay Console 菜单规划

## 当前菜单结构

```
总览
数据库实例
安全管理 > API Key
帮助 > 使用指南
```

## 目标菜单结构

```
数据管理
  总览
  数据库实例
  SQL 编辑器        ← 从数据库详情提升为一级入口
  数据导入          ← 独立入口，降低发现门槛

监控运维
  监控面板          ← 新增：连接数、存储趋势、唤醒次数等
  操作日志          ← 从总览表格独立出来

安全管理
  API Key          (已有)

账户
  用量与计费        ← 新增：计算时长、存储用量、费用概览
  账户设置          ← 新增：修改密码、个人信息

帮助
  使用指南          (已有)
```

## 优先级

| 优先级 | 功能 | 说明 | 状态 |
|--------|------|------|------|
| P0 | 监控面板 | 连接数、存储趋势、唤醒延迟可视化 | 待开发 |
| P0 | 操作日志 | 独立页面，审计追踪 | 待开发（代码已有，需提升为一级） |
| P1 | SQL 编辑器 | 选库后直接写 SQL，最高频操作 | 待开发（数据库详情内已有） |
| P1 | 数据导入 | 独立入口，不用先进数据库详情 | 待开发（导入向导已有） |
| P2 | 用量与计费 | 计算时长、存储用量、费用概览 | 待开发（免费阶段可后做） |
| P2 | 账户设置 | 修改密码、个人信息 | 待开发 |
