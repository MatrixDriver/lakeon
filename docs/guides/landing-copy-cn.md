## 核心卖点（Hero Section）

### 数据在中国，合规无忧
部署在华为云北京四区，数据存储在华为云 OBS 对象存储。
不出境、不经过海外节点，满足等保和数据合规要求。

### 按需付费，休眠零成本
无连接时自动休眠释放算力，数据安全持久化在对象存储。
有连接到来时 3 秒内自动唤醒，像打开水龙头一样简单。

### 数据库分支，开发提速
像 Git 一样给数据库创建分支。零拷贝、零成本。
在分支上做实验，不影响生产数据。

### 极简接入
标准 PostgreSQL 协议，兼容所有 PG 生态工具。
Prisma、Django、Spring Boot、psql — 改个连接串就能用。
