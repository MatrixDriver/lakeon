Get Started

一条命令装好，
三分钟内跑通。

echomem 是本地 daemon——不需要注册、不需要云服务、不需要 API key。装完就能给 Claude Code 提供记忆。

快速上手

安装与启动

1

安装

克隆仓库，创建虚拟环境，pip 安装。

git clone https://github.com/MatrixDriver/lakeon.git
cd lakeon/echomem
python3 -m venv .venv && source .venv/bin/activate
pip install -e ".[dev]"
2

初始化

创建 ~/.echomem/ 目录，包含 SQLite 数据库、blob 存储和配置。

echomem init
3

启动 daemon

FastAPI 进程监听 127.0.0.1:8473。

echomem start
# 或者自定义端口: ECHOMEM_PORT=9473 echomem start
4

检查状态

echomem status
# → daemon running at http://127.0.0.1:8473 · 5 memories · 3 agents
CLI

命令行写入与查询

daemon 启动后，CLI 通过 HTTP 与 daemon 通信。

# 写入记忆
echomem mem ingest "today I met jacky to discuss the storage driver" --agent cli

# 语义召回
echomem mem recall "storage driver"

# 列出最近记忆
echomem mem list --limit 10

# 查看详情
echomem mem get <memory_id>

# 删除
echomem mem delete <memory_id>
MCP

接入 Claude Code

在 ~/.claude/settings.json 的 mcpServers 中加入以下配置，重启 Claude Code 即生效。

{
  "mcpServers": {
    "echomem": {
      "command": "echomem-mcp-shim",
      "args": [],
      "env": {}
    }
  }
}

接入后，Claude Code 中可用 mcp__echomem__memory_ingest、mcp__echomem__memory_recall 等 5 个 MCP 工具直接操作记忆。

API 参考

HTTP API 端点

所有端点 base URL: http://127.0.0.1:8473

方法路径说明
GET /health 健康检查，返回 daemon 状态和统计信息

Memory

方法路径说明
POST /memory/ingest 写入一条记忆，自动触发 embedding + 衍生物 pipeline
POST /memory/recall 语义搜索，RRF 混合检索（向量 + FTS5）
GET /memory/list 分页列出记忆，支持按 agent 过滤
GET /memory/{id} 获取单条记忆详情
DELETE /memory/{id} 删除记忆

Derivatives

方法路径说明
GET /derivatives/timeline 时间流视图：按 Agent + 时间聚合事件
GET /derivatives/tree 分层摘要树：L0/L1/L2 三级摘要
GET /derivatives/graph 知识图谱：三元组子图查询，支持 hops
GET /derivatives/skills 程序性记忆：按上下文 surface 相关 skill

Context API · Phase 3 进行中

方法路径说明
POST /context/add_url 抓取 URL 内容，解析后存入 blob store + 进 pipeline
GET /context/ls 按前缀列出 blob 文件
GET /context/read 读取 blob 内容
POST /context/write 直接写入文本内容到 blob store
POST /context/mv 移动/重命名 blob 路径别名

Skills

方法路径说明
POST /skills/import 从 ~/.claude/skills import frontmatter
文档站

本地预览文档站

本文档站本身也可以本地跑——和线上完全一致的 Caddy + 静态文件。

cd lakeon/echomem/docs
docker build -t echomem-docs:dev .
docker run --rm -p 8080:8080 echomem-docs:dev
# 浏览器打开 http://localhost:8080/
